#### GAN Experiments on Fashion-MNIST

A COMP4211 Machine Learning project studying how different training and architecture choices affect GAN image generation quality on Fashion-MNIST.

The project includes:

- multiple controlled GAN experiments
- changes to latent dimension, model architecture, learning rate, training epochs, and generator loss
- Fréchet Inception Distance (FID) evaluation
- qualitative inspection of generated samples
- ablation-style analysis on the sensitivity of GAN training to generator-discriminator balance
